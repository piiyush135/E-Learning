Conditional Statements

Conditional statements allow you to execute different blocks of code based on specified conditions. In Python, you can use if, elif, and else statements for this purpose.

The if statement evaluates a condition and executes a block of code if the condition is true. The elif statement allows you to check additional conditions, and the else statement provides a fallback option if none of the conditions are met.

Loops

Loops are used to repeat a block of code multiple times. Python supports two types of loops: the for loop and the while loop.

The for loop iterates over a sequence of items, such as lists, strings, or ranges. It executes a block of code for each item in the sequence.

The while loop repeats a block of code as long as a specified condition is true. It continues executing the code until the condition becomes false.
