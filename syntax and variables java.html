Syntax and Variables in Java

Java is a powerful programming language used for developing a wide range of applications, from mobile apps to large-scale enterprise systems. Understanding the syntax and variables in Java is essential for writing clean and efficient code.

In this section, we'll cover the basic syntax of Java and how to declare and use variables.

Java Syntax

Java syntax refers to the rules and conventions used to write Java code. Here are some key points:

  • Declaration of Classes: Java programs are organized into classes, which contain methods and variables.
  • Curly Braces: Curly braces ({}) are used to define blocks of code, such as methods and classes.
  • Semicolons: Semicolons (;) are used to terminate statements in Java.
  • Comments: Java supports single-line (//) and multi-line (/* */) comments to add explanations and documentation to the code.

Variables

Variables are used to store data in a Java program. Each variable has a data type and a name. Here are some common data types in Java:

  • int: Used to store integer values (e.g., 10, -5).
  • double: Used to store floating-point values (e.g., 3.14, -0.5).
  • char: Used to store single characters (e.g., 'A', 'x').
  • boolean: Used to store true or false values.

Variables must be declared before they can be used. Here's the syntax for declaring a variable:

                
                    // Syntax: dataType variableName;
                    int age;
                    double salary;
                    char grade;
                    boolean isStudent;

After declaring a variable, you can assign a value to it using the assignment operator (=). For example:

                
                    // Assigning values to variables
                    age = 25;
                    salary = 30000.50;
                    grade = 'A';
                    isStudent = true;
