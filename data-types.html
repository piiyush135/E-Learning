Numeric Types

Python supports several numeric types, including integers, floating-point numbers, and complex numbers. These types are used to represent different kinds of numerical data.

Integers are whole numbers without any decimal points. They can be positive or negative. Example: 5, -10, 100.

Floating-point numbers are numbers with a decimal point or an exponent notation. They can represent both integer and fractional parts. Example: 3.14, -0.001, 2.0e-5.

Complex numbers have a real and imaginary part. They are written in the form a + bj, where a is the real part and b is the imaginary part. Example: 3 + 4j, -2 - 5j.

String Type

Strings are sequences of characters, enclosed within single or double quotes. Python provides several methods for working with strings, such as concatenation, slicing, and formatting.

Strings in Python are immutable, meaning they cannot be changed after they are created. However, you can create new strings from existing ones. Example: 'hello', "world", 'Python'.

String literals can span multiple lines using triple quotes. Example:

                '''This is a
                multi-line
                string'''

Boolean Type

Boolean values represent truth values, either True or False. They are used in logical operations, comparisons, and control flow statements.

Python supports boolean operations such as AND, OR, and NOT, as well as comparison operators like ==, !=, <, >, <=, and >=.

Boolean values are the result of comparison operations. Example: 5 == 5 (True), 10 < 5 (False).
