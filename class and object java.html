Classes and Objects in Java

In Java, a class is a blueprint for creating objects. It defines the properties and behaviors (methods) that objects of the class will have. An object is an instance of a class, and it represents a real-world entity.

Classes and objects are fundamental concepts in object-oriented programming (OOP), which is a programming paradigm based on the concept of "objects," which can contain data and code: data in the form of fields (often known as attributes or properties), and code in the form of procedures (often known as methods).

Defining a Class in Java

To define a class in Java, you use the class keyword followed by the class name. Here's a simple example:

                
                    public class Car {
                        // Fields (attributes)
                        String make;
                        String model;
                        int year;

                        // Methods
                        public void drive() {
                            System.out.println("The car is driving.");
                        }
                    }

In this example, we've defined a class named Car with three fields (make, model, and year) and one method (drive()).

Creating Objects

To create an object of a class in Java, you use the new keyword followed by the class name, followed by parentheses. Here's how you can create an object of the Car class:

                
                    Car myCar = new Car();

This statement creates a new Car object and assigns it to the variable myCar. Now, myCar is an instance of the Car class, and you can access its fields and methods using dot notation.

Accessing Fields and Methods

You can access the fields and methods of an object using dot notation. Here's an example:

                
                    myCar.make = "Toyota";
                    myCar.model = "Camry";
                    myCar.year = 2022;

                    myCar.drive(); // Calling the drive() method

This code sets the make, model, and year fields of the myCar object and then calls the drive() method to make the car drive.
