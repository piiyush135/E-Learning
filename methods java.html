Methods in Java

Methods, also known as functions, are blocks of code that perform specific tasks and can be called or invoked to execute their functionality. They help in organizing code, improving reusability, and making programs easier to understand.

In Java, methods are defined within classes and can be either instance methods or static methods.

  • Instance Methods: These methods belong to an instance of a class and can access instance variables and other instance methods. They are invoked using an object of the class.
  • Static Methods: These methods belong to the class itself rather than to any specific instance. They can be invoked using the class name and are often used for utility functions or operations that do not require access to instance variables.

Methods consist of a method signature and a method body:

  • Method Signature: It includes the method name, return type, and parameter list (if any).
  • Method Body: It contains the code that defines the functionality of the method.

Defining Methods

To define a method in Java, you use the following syntax:

                
                    access_modifier return_type method_name(parameter_list) {
                        // Method body
                    }

Here's what each part of the method definition represents:

  • Access Modifier: It specifies the visibility of the method (e.g., public, private, protected).
  • Return Type: It specifies the type of value that the method returns. If the method does not return any value, the return type is void.
  • Method Name: It is the name of the method, which is used to call or invoke the method.
  • Parameter List: It contains the list of parameters (if any) that the method accepts. Parameters are variables that store the values passed to the method when it is called.

Example: Method with Parameters

Here's an example of a method that takes two parameters (integers) and returns their sum:

                
                    public int add(int num1, int num2) {
                        int sum = num1 + num2;
                        return sum;
                    }

In this example, the method name is add, the return type is int, and it takes two parameters of type int (num1 and num2). Inside the method body, the sum of the two numbers is calculated and returned.
