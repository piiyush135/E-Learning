Introduction to Graphs

A graph is a mathematical structure consisting of vertices (also called nodes or points) and edges (also called links or lines) that connect pairs of vertices. Graphs are used to represent relationships between objects.

Here are some key concepts related to graphs:

  • Vertices (Nodes): These are the entities in the graph, represented as points.
  • Edges (Links): These are the connections between vertices, represented as lines or arcs.
  • Directed Graph: A graph in which the edges have a direction associated with them, indicating a one-way relationship between vertices.
  • Undirected Graph: A graph in which the edges do not have a direction associated with them, indicating a two-way relationship between vertices.
  • Weighted Graph: A graph in which each edge has a numerical value associated with it, called a weight, representing some kind of cost or distance.
  • Adjacency: Two vertices are said to be adjacent if there is an edge connecting them.
  • Degree: The degree of a vertex is the number of edges incident to it.

Graphs are widely used in computer science, mathematics, and various other fields to model various real-world scenarios such as social networks, transportation networks, computer networks, and more. They have a wide range of applications in algorithms, data structures, optimization, and network analysis.

Types of Graphs

Undirected Graph

An undirected graph is a graph in which edges have no direction. That is, the edge (u, v) is identical to the edge (v, u).

Directed Graph (Digraph)

A directed graph is a graph in which edges have a direction. That is, the edge (u, v) is different from the edge (v, u).

Representations of Graphs

  • Adjacency Matrix: A two-dimensional array where the value at index (i, j) represents whether there is an edge between vertices i and j.
  • Adjacency List: A collection of lists where each list represents the vertices adjacent to a particular vertex.

Graph Traversal

  • Breadth-First Search (BFS): Explores all the vertices in a graph level by level.
  • Depth-First Search (DFS): Explores as far as possible along each branch before backtracking.

Graph Algorithms

  • Shortest Path Algorithms: Algorithms to find the shortest path between two vertices in a graph, such as Dijkstra's algorithm and Bellman-Ford algorithm.
  • Minimum Spanning Tree (MST) Algorithms: Algorithms to find a subset of edges that forms a tree that includes every vertex, where the total weight of all edges in the tree is minimized, such as Prim's algorithm and Kruskal's algorithm.
